Введение

В этом проекте проводится анализ рынка жилья в Москве. Основная цель - выявить факторы, влияющие на цены недвижимости, и создать модели для предсказания стоимости жилья. В проекте используются методы анализа данных, визуализации и машинного обучения.

Данные

Набор данных содержит информацию о:

Местоположении: район, расстояние до метро, адрес
Характеристиках недвижимости: площадь, количество комнат, этаж, тип здания, год постройки
Ценах: стоимость квартиры, цена за квадратный метр
Дополнительных параметрах: наличие балкона, лифта, парковки
Данные собраны из открытых источников и обработаны перед анализом.

Настройка окружения

Для запуска проекта необходимы:

Python 3.x
Jupyter Notebook
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
Установка зависимостей:

bash
Copy code
pip install pandas numpy matplotlib seaborn scikit-learn
Предобработка данных

Шаги предобработки данных:

Обработка пропущенных значений
Кодирование категориальных переменных
Нормализация числовых признаков
Разделение данных на обучающую и тестовую выборки
Исследовательский анализ данных

Исследовательский анализ данных (EDA) включает:

Описательная статистика: среднее, медиана, стандартное отклонение
Визуализация распределений: гистограммы, box plots, тепловые карты
Корреляционный анализ: матрица корреляции
Анализ тенденций: влияние площади, количества комнат, этажа на цену
Примеры визуализаций:

Распределение цен по районам Москвы
Зависимость цены от расстояния до метро
Влияние года постройки на стоимость жилья
Моделирование

Использованные алгоритмы:

Линейная регрессия
Решающие деревья
Случайный лес
Градиентный бустинг
Этапы моделирования:

Обучение моделей на обучающей выборке
Оценка моделей на тестовой выборке с использованием метрик R², RMSE, MAE
Настройка гиперпараметров для улучшения точности
Результаты

Результаты анализа включают:

Влияние различных характеристик на цену жилья
Модель с наилучшей производительностью и её метрики
Визуализация предсказанных и реальных цен
Ключевые выводы:

Местоположение и площадь жилья имеют наибольшее влияние на цену
Модели случайного леса и градиентного бустинга показали наилучшие результаты
Заключение

В заключении обобщаются результаты анализа и предлагаются направления для дальнейших исследований, такие как:

Анализ влияния экономических и социальных факторов
Прогнозирование изменений цен на жилье в будущем
Использование более сложных моделей, таких как нейронные сети
